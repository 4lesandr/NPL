# Lua - Извлечение email, телефонов и IP-адресов из текста

## Описание
Программа извлекает email адреса, номера телефонов и IP-адреса из текстового файла с помощью регулярных выражений Lua.

## Запуск
```bash
cd lua_text_parser
./run.sh input.txt output.json
```

## Альтернативный способ
```bash
lua text_parser.lua input.txt output.json
```

## Требования
Требуется Lua интерпретатор.
