# Haskell - Вычисление чисел Фибоначчи с мемоизацией

## Описание
Программа вычисляет числа Фибоначчи рекурсивно с использованием мемоизации для оптимизации.

## Компиляция и запуск
```bash
cd haskell_fibonacci
ghc -o fibonacci Fibonacci.hs
./fibonacci
```

## Альтернативный способ запуска
```bash
ghci Fibonacci.hs
> main
```

## Зависимости
Требуется GHC (Glasgow Haskell Compiler) и библиотека containers (обычно входит в стандартную установку).
