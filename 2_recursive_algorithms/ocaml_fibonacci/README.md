# OCaml - Вычисление чисел Фибоначчи с мемоизацией

## Описание
Программа вычисляет числа Фибоначчи рекурсивно с использованием мемоизации (Hashtbl) для оптимизации.

## Компиляция и запуск
```bash
cd ocaml_fibonacci
ocamlc -o fibonacci fibonacci.ml
./fibonacci
```

## Альтернативный способ запуска
```bash
ocaml fibonacci.ml
```

## Требования
Требуется OCaml компилятор (ocamlc или ocamlopt).
