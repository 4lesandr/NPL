# Rust - Параллельная обработка данных из файла

## Описание
Программа параллельно обрабатывает данные из файла, используя потоки Rust.

## Компиляция и запуск
```bash
cd rust_parallel_processing
cargo build --release
cargo run --release data.txt
```

## Пример файла данных (data.txt)
```
12345
67890
abc123def456
test789
```
