# Ruby - Преобразование CSV в JSON

## Описание
Программа преобразует CSV файл в JSON формат с использованием стандартных библиотек Ruby.

## Запуск
```bash
cd ruby_csv_to_json
./run.sh data.csv output.json
```

## Альтернативный способ
```bash
ruby csv_to_json.rb data.csv output.json
```

## Требования
Требуется Ruby (обычно предустановлен в macOS/Linux).
