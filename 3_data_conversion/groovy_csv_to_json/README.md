# Groovy - Преобразование CSV в JSON

## Описание
Программа преобразует CSV файл в JSON формат с использованием библиотеки Apache Commons CSV.

## Запуск
```bash
cd groovy_csv_to_json
./run.sh data.csv output.json
```

## Альтернативный способ
```bash
groovy csv_to_json.groovy data.csv output.json
```

## Требования
Требуется Groovy. Зависимости загружаются автоматически через @Grab.
