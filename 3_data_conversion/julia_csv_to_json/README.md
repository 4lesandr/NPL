# Julia - Преобразование CSV в JSON

## Описание
Программа преобразует CSV файл в JSON формат. Упрощенная версия без внешних зависимостей, использует только стандартную библиотеку Julia.

## Запуск
```bash
cd julia_csv_to_json
./run.sh data.csv output.json
```

## Альтернативный способ
```bash
julia csv_to_json.jl data.csv output.json
```

## Требования
Требуется только Julia (без дополнительных пакетов).
